### File View Flask Vue
Just use python + flask + vue (vite). Don't have any library.
![Program previw](https://github.com/treeloys/fileviewflaskvue/blob/main/docs/Screenshot_1.jpg?raw=true)

### How to setup?
Only windows testing
##### Install
For python

`pip install flask`
`pip install flask-cors`

For vue (vite)

`cd picinity-front`
`npm install`
##### Run
Backend

`set FLASK_DEBUG=1`
`python.exe main.py `

Front

`npm run dev`
